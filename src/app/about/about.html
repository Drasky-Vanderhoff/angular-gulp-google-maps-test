How to use this thing For the Greater Good!

Adding a new address

  1. Click on Add New Address.
    It will show a modal where you can add as many address as you want.
  2. Once you added all those address, close the modal window.
  3. After that can move the markers that were created to place them whenever you want.
  4. After that you can confirm each marker's position separately using the OK button
